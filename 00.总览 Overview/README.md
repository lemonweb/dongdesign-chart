## 1. 目录定位

本目录是数据可视化设计体系的入口，用于说明整套 Wiki 如何使用、AI 如何读取、图表如何初步选型。

它不写具体图表规范，只负责提供全局导航。

## 2. 使用时机

以下情况应优先读取本目录：

- 第一次使用本 Wiki
- 不确定该先读哪个文件
- 不确定当前需求适合什么图表
- 需要让 AI 按统一流程工作

## 3. 读取顺序

建议按以下顺序读取：

1. `01.使用说明 How-to-use.md`
2. `02.AI读取顺序 AI-reading-flow.md`
3. `03.图表选型决策树 Decision-tree.md`
4. `04.术语表 Glossary.md`
5. `05.Codex记忆迁移 Codex-memory-migration.md`

## 4. 不在本目录处理的内容

本目录不处理具体设计细节。

- 图表样式规则：查阅 `01.视觉语言 Design-language/`
- 图表组件规范：查阅 `02.图表模式 Chart-type/`
- 业务页面组合：查阅 `03.业务场景 Pattern/`
- G2 / ECharts 实现：查阅 `04.引擎适配 Adaptation/`
- AI 生成约束：查阅 `05.执行规则 AI-rules/`
