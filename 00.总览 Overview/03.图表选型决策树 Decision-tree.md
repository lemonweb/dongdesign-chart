# 图表选型决策树 Decision Tree

> 快速判断当前需求应该优先进入哪类图表文档。  
> 本文件负责“先去哪”，不负责完整选型细则。

---

# 1. 文档目的

本决策树用于帮助 AI 与设计师快速完成：

```txt
业务问题
→ 图表方向判断
→ 候选图表家族
→ 下一步查阅文档
```

目标：
在最短路径内找到正确图表方向。

---

# 2. 使用方式

按顺序判断：

```txt
Step 1：业务目标
Step 2：数据结构
Step 3：展示空间
Step 4：候选图表家族
Step 5：进入具体组件文档
```

如果出现多意图场景：

继续查阅：

```txt
02.Chart-pattern/
└── Chart-selection-rules.md
```

---

# 3. 一级决策：业务目标（Primary Intent）

先判断：

用户最想回答什么问题？

---

## A. 看比较（Comparison）

典型问题：

```txt
谁更高？
谁更低？
差多少？
```

→ 优先进入：

```txt
Basic-charts/
Composite-charts/
```

候选：

- 柱状图
- 条形图
- 分组柱状图
- 堆叠柱状图

---

## B. 看趋势（Trend）

典型问题：

```txt
随时间怎么变化？
上涨还是下降？
是否异常？
```

→ 优先进入：

```txt
Basic-charts/
Composite-charts/
KPI-charts/
```

候选：

- 折线图
- 面积图
- 堆叠面积图
- Sparkline

---

## C. 看占比（Proportion）

典型问题：

```txt
占整体多少？
构成如何？
```

→ 优先进入：

```txt
Basic-charts/
Composite-charts/
```

候选：

- 环图
- 饼图
- 百分比堆叠柱状图
- Treemap

---

## D. 看分布（Distribution）

典型问题：

```txt
数据集中在哪？
有没有异常？
是否离散？
```

→ 优先进入：

```txt
Statistical-charts/
```

候选：

- 直方图
- 箱线图
- 散点图
- 热力图

---

## E. 看关系（Relationship）

典型问题：

```txt
变量之间是否相关？
```

→ 优先进入：

```txt
Statistical-charts/
```

候选：

- 散点图
- 气泡图
- 热力图

---

## F. 看流程 / 转化（Flow）

典型问题：

```txt
在哪一步流失？
来源去哪？
```

→ 优先进入：

```txt
Flow-charts/
```

候选：

- 漏斗图
- 桑基图
- 路径流转图

---

## G. 看地域（Geo）

典型问题：

```txt
不同地区表现如何？
```

→ 优先进入：

```txt
Geo-charts/
```

候选：

- 区域地图
- 气泡地图
- 迁徙图

---

## H. 看指标状态（Monitoring / KPI）

典型问题：

```txt
当前状态如何？
完成率多少？
```

→ 优先进入：

```txt
KPI-charts/
```

候选：

- 指标卡
- 迷你趋势图
- 进度条
- 仪表盘

---

# 4. 二级决策：数据结构

在图表方向确定后，再判断字段结构。

---

## 4.1 是否有时间字段？

```txt
YES → 趋势类优先
NO  → 比较 / 分布 / 关系类优先
```

---

## 4.2 是否有 category（类目字段）？

```txt
YES → 柱状图 / 条形图 / 饼图 / 堆叠图
NO  → 散点 / 关系 / 密度图
```

---

## 4.3 是否有 source / target？

```txt
YES → Flow-charts/
```

---

## 4.4 是否有 geo？

```txt
YES → Geo-charts/
```

---

## 4.5 是否有 hierarchy（层级结构）？

```txt
YES → Treemap / Sunburst / Tree
```

---

## 4.6 是否是多变量（>=3 numeric）？

```txt
YES → 气泡图 / 雷达图 / 热力图
```

继续查：

```txt
Chart-capability-matrix.md
```

---

# 5. 三级决策：展示空间

---

## 5.1 小卡片 / KPI 区域

优先：

- KPI-chart
- Sparkline
- Mini Bar
- Ring

避免：

- 桑基图
- 地图
- 多轴复杂图

---

## 5.2 标准分析卡片

优先：

- 柱状图
- 折线图
- 堆叠图
- 散点图
- 漏斗图

---

## 5.3 大屏 / 监控

优先：

- 指标卡
- 趋势图
- 排名
- 地图
- 热力图

---

## 5.4 移动端

优先：

- KPI
- 简化折线
- 条形图
- 环图

避免：

- 桑基图
- 雷达图
- 多系列复杂图

---

# 8. 禁止项

AI 禁止：

- 根据“图表名称”直接判断用途
- 看到占比就默认饼图
- 看到趋势就默认折线图
- 没有 geo 仍然使用地图
- 没有 source/target 使用桑基图
- 因视觉丰富选择复杂图表
- 跳过组件文档
- 跳过能力矩阵
- 跳过 Chart-selection-rules.md
