# 区间标记 Mark Area

## 1. 文档定位

本文定义活动区间、异常区间、阶段背景、风险区间、预测区间和业务窗口的使用规则。

## 2. 区间类型

| 类型 | 使用条件 | 典型场景 |
|---|---|---|
| 活动区间 | 有明确开始和结束时间 | 活动复盘、营销投放 |
| 异常区间 | 有异常规则或异常检测结果 | 指标监控、质量分析 |
| 阶段背景 | 数据天然分阶段 | 漏斗、流程、生命周期 |
| 风险区间 | 有风险等级或阈值范围 | 风控、履约、库存 |
| 预测区间 | 有预测范围或置信区间 | 预测图、达成风险 |

## 3. 样式继承

- 区域标注填充读取 `chart/mark-area/fill/default`。
- 标注层级低于主数据系列，高于网格线。
- 区间标签继承 `../01-design-language/label.md` 的区间标签规则。

## 4. 使用规则

- 区间必须绑定明确起止范围。
- 区间文案应说明业务含义，不只写“阶段 1”。
- 区间背景透明度应克制，不得影响主数据读数。
- 多个区间相邻时，优先使用分隔线、顶部标签或图例说明，避免大面积彩色背景堆叠。
- 对时间序列图，区间边界必须与 X 轴时间刻度一致。

## 5. 禁止项

- 禁止把区间背景当作装饰色块。
- 禁止在没有正常范围规则时标记“异常区间”。
- 禁止用区间遮挡 Tooltip、数据标签和主趋势路径。
