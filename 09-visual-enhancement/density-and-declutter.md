# 密度控制与降噪 Density and Declutter

## 1. 文档定位

本文定义密集图表中的标签隐藏、抽样、折叠、聚合、避让和信息降噪规则。

## 2. 优先级

当信息过密时，按以下顺序处理：

```txt
隐藏普通标签
→ 保留关键点
→ 调整方向或位置
→ 使用 Tooltip
→ 抽样 / 聚合 / 合并 Others
→ 筛选 / 缩放 / 下钻
→ 更换图表类型
```

## 3. 常见场景

| 场景 | 降噪方式 |
|---|---|
| 时间点过密 | 保留首尾和固定间隔标签，精确值进入 Tooltip |
| 类目过多 | Top N + Others、横向条形、搜索或筛选 |
| 系列过多 | 默认展示重点系列，图例筛选，小多图 |
| 点位过多 | 透明度、聚合、框选、缩放、密度图 |
| 标签重叠 | 隐藏普通标签，保留关键点，引导线避让 |
| Tooltip 条目过多 | 折叠、分页、只显示重点系列 |

## 4. Agent 规则

- 不得通过缩小字号到 `12px` 以下解决拥挤。
- 不得为了塞下内容压缩图表主体到不可读。
- 优先保留与叙事重点相关的信息。
- 降噪后必须保证 Tooltip 或明细入口能补齐完整读数。
