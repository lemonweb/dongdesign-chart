# 图表类型增强映射 Chart Type Mapping

## 1. 文档定位

本文定义不同图表类型适合使用的增强方式、谨慎使用方式和禁用方式。

具体样式仍以 `../02-chart-type/` 对应图表文档为准。

## 2. 映射表

| 图表类型 | 推荐增强 | 谨慎使用 | 禁止 / 避免 |
|---|---|---|---|
| 折线图 | 最新值、峰谷、异常点、目标线、活动区间、事件标注 | 多条参考线、全部点标签 | hover 指示线画成虚线 |
| 面积图 | 阶段区间、趋势说明、关键点 | 多系列面积高亮 | 面积遮挡后仍强行叠加标签 |
| 柱状图 / 条形图 | Top / Bottom、差值、目标线、类目 hover | 每柱不同色 | 随机配色表达普通类目 |
| 饼图 / 环图 | 主分类外置标签、Others 合并 | 过多外置标签 | 用饼图表达趋势 |
| 散点图 | 异常点、象限、趋势线、框选 | 全部点标签 | 把相关性写成因果 |
| 气泡图 | 异常气泡、规模说明、框选摘要 | 过多颜色和大小分组 | 用气泡面积表达精确值 |
| 直方图 | 阈值线、异常区间、Brush 区间 | 多条阈值线 | 每个 bin 随机上色 |
| 热力图 | 高低区间、当前单元格、行列辅助 | 过多文字标签 | 破坏色阶方向 |
| 漏斗图 | 阶段高亮、转化率标注、流失说明 | 过多阶段 | 用装饰箭头替代转化关系 |
| 桑基图 | 路径高亮、节点联动、来源去向 | 全路径同时高亮 | 忽略宽度编码语义 |
| 柱线混合图 | 目标线、类目高亮、折线 hover 点 | 双轴 callout | 同时显示类目高亮和垂直 line 指示器 |
| 指标卡 | 目标差、趋势缩略、状态标签 | 过多解释文本 | 无目标时判断达标 |
| 表格 | 排序、条件高亮、进度条、异常行 | 大面积色阶 | 只靠颜色表达风险 |

## 3. 使用规则

- 图表类型文档已有增强规则时，优先引用具体图表文档。
- 图表类型未覆盖的增强，只能作为通用建议，并标记 `inferred`。
- 若增强需要跨图表联动，必须补充交互反馈和作用范围。
