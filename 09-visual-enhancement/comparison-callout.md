# 对比说明 Comparison Callout

## 1. 文档定位

本文定义同比、环比、目标差、竞品差、阶段差异和贡献差异的 callout 表达规则。

## 2. 使用条件

Callout 适合承接“差多少、为什么重要、下一步看哪里”的短判断。

| 场景 | 必要证据 | 推荐内容 |
|---|---|---|
| 同比 / 环比 | 当前值、对比期值、口径 | 变化方向、幅度、时间范围 |
| 目标差 | 当前值、目标值 | 差距、达成状态 |
| 阶段差异 | 两个阶段边界和值 | 阶段变化、事件背景 |
| 贡献差异 | 总变化、贡献拆解 | 主要贡献对象 |
| 竞品 / 同类差 | 同类对象和口径 | 相对位置、差距 |

## 3. 文案结构

```txt
发现：发生了什么
判断：是否值得关注
证据：用哪个数支持
下一步：可选的探索入口
```

图表内 callout 应尽量短，复杂解释放入洞察卡或 Tooltip note。

## 4. 样式规则

- 数值格式必须与坐标轴、Tooltip 和指标卡一致。
- 趋势方向色必须继承语义色规则，并注意业务正负含义可能反转。
- callout 不应使用大面积卡片遮挡图表主体。
- 多个 callout 同时存在时，优先保留最贴近叙事重点的一条。

## 5. 禁止项

- 禁止没有对比基准时写“提升 / 下降明显”。
- 禁止只写结论不展示证据。
- 禁止把 AI 推测原因写成确定事实。
