# 洞察标注 Insight Annotation

## 1. 文档定位

本文定义关键点、异常点、拐点、峰谷、最新值和业务事件标注的使用规则。

## 2. 适用场景

| 标注对象 | 使用条件 | 典型图表 |
|---|---|---|
| 最大值 / 最小值 | 用户需要识别峰谷或范围边界 | 折线图、柱状图、面积图 |
| 最新值 | 用户关注当前状态 | 折线图、面积图、指标卡 |
| 异常点 | 有阈值、基准或异常规则 | 折线图、散点图、气泡图 |
| 拐点 | 趋势方向明显变化且有足够时间序列 | 折线图、面积图 |
| 事件点 | 有明确业务事件时间 | 趋势图、活动复盘 |
| Top / Bottom | 排序或优先级是核心问题 | 柱状图、条形图、表格 |

## 3. 样式继承

- 字号、字重和标签距离继承 `../01-design-language/label.md`。
- 关键数据标签使用 `.numberSmall 12/18 600`。
- 说明型标签使用 `.textSmall 12/18 400`。
- 引导线使用 `1px`，绑定 `chart/label/leader-line/default` 或弱化结构色。
- 折线 hover markPoint 固定 `8 × 8px`，不得为了强调放大。

## 4. 内容结构

标注文案应尽量短，并包含必要语义：

```txt
对象 / 时间
关键值
原因或状态（可选）
```

示例结构：

```txt
峰值 5.8 万
低于目标 12%
活动上线
异常商品 A
```

## 5. 布局规则

- 标签优先放在点位外侧，距离点位 `8-12px`。
- 靠近边界时向图内偏移。
- 标注不得遮挡主趋势线、柱体、图例和 Tooltip。
- 密集点位优先只标 `1-3` 个关键点，其余进入 Tooltip。
- 散点图和气泡图默认不显示全部点位标签，只标异常或选中点。

## 6. 禁止项

- 禁止把所有数据点都标成洞察。
- 禁止没有依据地标注“异常、风险、原因”。
- 禁止用标注替代坐标轴、图例或 Tooltip。
- 禁止在同一位置堆叠多个 callout。
