# 高亮与弱化 Highlight

## 1. 文档定位

本文定义数据点、系列、类别、区间和状态的高亮、弱化、选中、hover 与聚焦策略。

## 2. 高亮类型

| 类型 | 用途 | 常见触发 |
|---|---|---|
| 静态重点高亮 | 强调 Top、异常、目标差距等重点 | 页面加载后默认 |
| hover 高亮 | 帮助读取当前对象细节 | 鼠标悬停 |
| 选中高亮 | 表达当前分析对象或筛选对象 | 点击、框选 |
| 联动高亮 | 表达跨图表关联对象 | 选择某图表对象后 |
| 弱化 | 降低非重点对象干扰 | hover、选中、筛选 |

## 3. 基础规则

- 高亮只能改变强调程度，不应改变业务语义。
- 高亮颜色应继承系列色、语义色或主题变量，不得随机配色。
- 非重点弱化不得使数据不可读；弱化后仍应能通过 Tooltip 读取。
- 只靠颜色不足以表达异常或风险，必须配合标签、符号或说明。

## 4. 图表规则

| 图表 | 推荐高亮 | 注意 |
|---|---|---|
| 折线 / 面积 | 当前点、当前系列、关键点 | hover 指示线必须是垂直 `1px` 实线 |
| 柱状 / 柱线混合 | 完整类目占位高亮矩形 | 不得同时显示垂直 line 指示器 |
| 散点 / 气泡 | 当前点、选中点、异常点 | 未选中点可降透明度，不默认给所有点描边 |
| 漏斗 | 当前阶段、相邻转化带 | 不得破坏阶段递减色阶 |
| 热力图 | 当前单元格、行列辅助 | 色阶含义必须保持稳定 |

## 5. 图层规则

```txt
Tooltip / hover label
→ hover 点 / 选中描边
→ 主数据系列
→ 类目高亮矩形 / 弱化遮罩
→ 网格线
```

高亮矩形、选中轮廓和弱化遮罩应放在 `interactionLayer`，不得混入 `seriesLayer`。
