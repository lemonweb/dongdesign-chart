# 参考线 Reference Line

## 1. 文档定位

本文定义目标线、阈值线、均值线、基准线、预警线和达标线的使用规则。

## 2. 使用条件

| 类型 | 使用条件 | 支持判断 |
|---|---|---|
| 目标线 | 有明确目标、计划或预算 | 是否达标、差距多少 |
| 阈值线 | 有 SLA、风控、预警或业务边界 | 是否异常、是否越界 |
| 均值线 | 需要与整体平均水平对比 | 是否高于 / 低于平均 |
| 基准线 | 有历史、同类或行业基准 | 是否偏离基准 |
| 预测线 | 有模型预测或计划曲线 | 实际与预测偏差 |

## 3. 样式继承

- 参考线 token 读取 `chart/mark-line/stroke/default`。
- 警戒线可引用 semantic warning / danger。
- 参考线标签读取 `chart/mark-line/label/default`。
- 带背景标签读取 `chart/mark-line/label/background`。
- 参考线层级低于主数据系列，高于网格线。

## 4. 使用规则

- 参考线必须说明业务含义，例如“目标 GMV”“SLA 48h”“历史均值”。
- 标签应靠近线条端点或不遮挡数据的位置。
- 多条参考线超过 `3` 条时，应改用区间、筛选或说明列表。
- 目标线、预测线、网格线和 hover 指示线必须有不同语义，不得混用样式。
- 折线 hover 指示线不是参考线，必须遵守 `../01-design-language/tooltip.md` 和 `axis.md`。

## 5. 禁止项

- 禁止没有目标或阈值字段时绘制目标线 / 阈值线。
- 禁止用品牌色或高饱和色绘制普通均值线。
- 禁止参考线压过主数据系列。
- 禁止只画线不写含义。
