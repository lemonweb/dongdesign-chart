# 标签 Label

## 1. 文档定位

本文定义图表数据标签、关键点标签、阈值标签和标注引导线的使用规则。

它回答：

> 图表中什么时候应该直接显示文字标签？标签应该显示什么、放在哪里、用什么样式，以及什么时候必须隐藏？

本文不重新定义颜色和字体变量。颜色、字号和字重应读取：

- `02.主题变量 Theme-token.md`
- `03.色彩系统 Color.md`
- `04.字体排版 Typography.md`

## 2. 标签类型

| 类型 | 用途 | 典型图表 |
|---|---|---|
| 数据标签 | 显示某个数据点或图形的数值 | 柱状图、折线图、饼图 |
| 关键点标签 | 标记最大值、最小值、最新值、异常点 | 折线图、散点图 |
| 阈值标签 | 说明目标线、预警线、基准线 | 折线图、柱状图 |
| 区间标签 | 说明活动区间、异常区间、阶段 | 折线图、面积图 |
| 引导标签 | 通过引导线连接图形与文字 | 饼图、环图、密集标注 |

## 3. 使用原则

标签只用于增强图表主体表达，不替代坐标轴和 Tooltip。

使用规则：

- 默认不为每个数据点显示标签。
- 趋势图优先标记最大值、最小值、最新值或异常点。
- 柱状图优先标记 Top / Bottom、目标差异或异常值。
- 漏斗图可标记阶段值和阶段转化率。
- 饼图 / 环图可标记主要分类名称和占比。
- 标签密集时优先隐藏，精确值放入 Tooltip。

禁止项：

- 禁止铺满所有点位导致遮挡趋势。
- 禁止标签遮挡折线、柱体、坐标轴、图例或 Tooltip。
- 禁止只用颜色表达异常，必须配合文字或符号。
- 禁止为了塞下标签使用小于 `12px` 的字号。

## 4. 样式规则

| 元素 | 样式 | 颜色 |
|---|---|---|
| 常规数据标签 | `.numberSmall` `12/18` `400` | `text/300` 或 `chart/label/text/default` |
| 关键数据标签 | `.numberSmall` `12/18` `600` | `text/400` 或对应系列色 |
| 说明型标签 | `.textSmall` `12/18` `400` | `text/300` |
| 阈值 / 预警标签 | `.textSmall` `12/18` `600` | 对应 `语义色板/*` |
| 引导线 | `1px` | `chart/label/leader-line/default` 或弱化结构色 |
| 折线 hover markPoint | `8 × 8px` 圆形，`1px` 描边 | 填充绑定对应 `分类色板/item-*`，描边为白色 |

折线图关键标签可使用对应系列色，但必须满足：

- 只用于少量关键点。
- 标签与点位之间保持可读距离。
- 不改变该系列的业务含义。

## 5. 布局规则

### 5.1 折线图

- 最新值标签优先放在最后一个点的右上方或右侧。
- 最大值 / 最小值标签优先放在点位外侧，避开折线路径。
- 标签距离点位建议 `8-12px`。
- 如果标签靠近图表边界，应向图内侧偏移。
- hover 状态下的点位值优先由 Tooltip 承载，不重复显示静态标签。
- hover markPoint 固定为 `8 × 8px` 圆形，系列色填充，`1px` 白色描边；不得放大为 12px、14px 或使用粗描边制造强调。

### 5.2 柱状图

- 正值标签优先放在柱体外侧上方或柱体内部顶部。
- 负值标签优先放在柱体外侧下方。
- 标签应与柱体中心对齐。
- 柱体过短时标签放在外侧，避免挤压。

### 5.3 饼图 / 环图

- 主要分类可显示外置标签。
- 外置标签应使用引导线连接扇区。
- 分类过多时只标 Top 分类，其余进入 Tooltip 或图例。

## 6. 重叠处理

标签重叠时按以下优先级处理：

```txt
隐藏普通标签
→ 保留关键点标签
→ 调整标签方向
→ 使用引导线
→ 移入 Tooltip
```

AI 生成图表时必须输出标签策略：

| 字段 | 示例 |
|---|---|
| 是否显示标签 | `true / false` |
| 标签对象 | `最大值、最小值、最新值` |
| 标签样式 | `.numberSmall 12/18 600` |
| 标签颜色 | `分类色板/item-1` 或 `text/300` |
| 重叠策略 | `overlap hidden` |

## 7. 与主题变量的关系

| Token | 用途 |
|---|---|
| `chart/label/text/default` | 常规数据标签 |
| `chart/label/text/secondary` | 次级标签 |
| `chart/label/background/default` | 带背景标签 |
| `chart/label/leader-line/default` | 饼图、环图等引导线 |
| `chart/label/overlap/hidden` | 标签重叠隐藏策略 |

当上述 token 在工具中暂未存在时，应使用已定义的文字、系列色和结构色 token 作为降级方案，并在验证清单中标记缺口。

## 8. AI 自查清单

- 标签是否只服务关键读数？
- 是否避免为所有点默认显示标签？
- 标签是否使用 `12/18` 的固定排版？
- 数字格式是否和 Tooltip / 坐标轴一致？
- 标签是否遮挡数据图形或其他组件？
- 关键标签是否能解释为什么需要突出？
- 标签缺失时 Tooltip 是否能提供完整读数？
