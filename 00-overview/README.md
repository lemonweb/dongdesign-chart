## 1. 目录定位

本目录是数据可视化设计体系的入口，用于说明整套 Wiki 如何使用、AI 如何读取、图表如何初步选型。

它不写具体图表规范，只负责提供全局导航。

## 2. 使用时机

以下情况应优先读取本目录：

- 第一次使用本 Wiki
- 不确定该先读哪个文件
- 不确定当前需求适合什么图表
- 需要让 AI 按统一流程工作

## 3. 读取顺序

建议按以下顺序读取：

1. `../ai-file-index.md`
2. `how-to-use.md`
3. `ai-reading-flow.md`
4. `00-overview/decision-tree.md`
5. `glossary.md`

## 4. 不在本目录处理的内容

本目录不处理具体设计细节。

- 图表样式规则：查阅 `../01-design-language/`
- 图表组件规范：查阅 `../02-chart-type/`
- 业务页面组合：查阅 `../03-pattern/`
- G2 / ECharts 实现：查阅 `../04-adaptation/`
- AI 生成约束：查阅 `../05-rules/`
- 可视化增强：查阅 `../09-visual-enhancement/`
