# 表格组件规范（Table）

> 提取自 Figma 表格枚举样例（node 39:8471）。表格由「Table表格列」按列拼装。颜色/字体引用《color-tokens.md》《typography.md》。当前先枚举常见列类型，后续可补充。

---

## 一、整体规格

| 项 | 值 |
|---|---|
| 表格宽 | 对齐内容列（示例 912） |
| 表头行高 | 40px |
| 数据行高 | ~48px |
| 单元格水平内边距 | 16px（`spacing-horizontal-m`） |
| 行分隔线 | 1px `--jd-color-line-100`（#E6E7EB），每行底部 |
| 表头 | 文字 14/22 **Semibold** `text-400`；底色白或浅灰 `fill-100`（#F7F8FA） |
| 表格圆角 | 8px；外可描边 `line-100`；浮层态用 `shadow-middle` |
| 首列 | 可吸附（`position:sticky; left:0`），窄屏整表横向滚动 |
| 行 hover | 浅灰高亮 `fill-100` |
| 字体 | 中文 PingFang SC；**数字 Roboto** |

## 二、列类型（逐类样式）

### 1. 文本列 Text
- 对齐：**左对齐**
- 样式：14/22 Regular；主体文本 `text-400`（#1C1D1F），次要信息可用 `text-300`（#515357）
- 例：`京东零售`

### 2. 金额列 Currency
- 对齐：**右对齐**
- 样式：数字字体 **Roboto** Regular 14；`text-400`；**¥ 前缀**；千分位分隔
- 例：`¥2,299`

### 3. 数值列 Number
- 对齐：**右对齐**
- 样式：Roboto Regular 14；`text-400`；千分位分隔
- 例：`2,299`

### 4. 对比列 Comparison（变化值）
- 对齐：**右对齐**
- 样式：Roboto Regular 14；带 `+/−` 号；**按产品涨跌规则上色**（`--jd-color-trend-*`）：对外红升绿降 / 对内绿升红降；持平灰 `trend-even`
- 例：`−43.10%`（绿，下降）/ `+43.10%`（红，上涨）— 示例为**对外口径**

### 5. 对比列（可排序）Comparison + Sort
- 同"对比列"，**表头带排序箭头 `⇅`**，点击切换升/降序（当前列高亮排序状态）
- 值颜色规则同上

### 6. 占比列 Proportion
- 结构：**进度条 + 百分比文字**（条在左、% 在右）
- 进度条：轨道 `fill-200`（#F2F4F7），填充 `--jd-color-info-normal`/brand（#3768FA），圆角 pill（999）
- 百分比：14 `text-400`，右侧
- 例：`▓▓▓░ 75%`

### 7. 状态列 Status
- 结构：**彩色圆点 + 文字**；圆点 ~8px 圆形，文字 14 `text-300`
- 状态色映射：

| 状态 | 圆点色 | 令牌 |
|---|---|---|
| 进行中 | 蓝 | `info-normal` #3768FA |
| 待审核 | 橙 | `warning-normal` #F08433 |
| 已通过 | 绿 | `success-normal` #00B26F |
| 已驳回 | 红 | `error-normal` #F33B50 |
| 已取消 | 灰 | `text-100` #B0B2B8 |

---

## 三、通用规则

- **数字类列（金额 / 数值 / 对比 / 占比%）一律右对齐**；文本、状态列左对齐；表头对齐与列内容一致。
- 数字统一用 **Roboto**，中文用 **PingFang SC**。
- 变化值颜色**不写死**，按产品涨跌规则（引用 `trend-*`）。
- 首列吸附、窄屏横向滚动（详见《report-layout.md》表格约定）。
- 每行底部 1px `line-100` 分隔；hover 行浅灰高亮。

## 四、令牌速查

| 用途 | 令牌 / 值 |
|---|---|
| 表头文字 | text-400 Semibold 14/22 |
| 单元格文本 | text-400 / text-300，Regular 14 |
| 数字 | Roboto Regular 14，右对齐 |
| 变化值 | trend-up/down/even（按产品） |
| 进度条 | 轨道 fill-200 / 填充 info #3768FA / 圆角 999 |
| 状态点 | info / warning / success / error / text-100 |
| 分隔线 / 表头底 | line-100 #E6E7EB / fill-100 #F7F8FA |
| 行高 / 表头高 / 内边距 | 48 / 40 / 16 |
