## 1. 目录定位

本目录用于管理单个图表组件的设计指南，例如柱状图、折线图、饼图、气泡图、雷达图、漏斗图等。

本目录不按单一功能给图表定性。图表可以同时表达比较、趋势、占比、关系、分布等多种能力。

## 2. 使用时机

以下情况应读取本目录：

- 需要选择合适图表
- 需要查看某个图表的适用场景
- 需要确认图表所需数据字段
- 需要生成单个图表的设计或代码
- 需要判断某个图表是否适合当前业务问题

## 3. 读取顺序

AI 选图时应按以下顺序读取：

1. `02-chart-type/capability-matrix.md`
2. `02-chart-type/selection-rules.md`
3. `02-chart-type/complexity-level.md`
4. 具体图表组件文档

## 4. 目录使用原则

- 目录只是图表归档方式，不代表图表只有单一用途。
- 图表选型必须结合能力矩阵、选型规则和具体图表文档。
- 每个图表文档应包含 metadata、适用场景、不适用场景、数据结构、视觉规则、交互规则和 AI warning。

## 5. 不在本目录处理的内容

- 通用视觉样式：查阅 `01-design-language/`
- 业务页面组合：查阅 `03-pattern/`
- G2 / ECharts 代码映射：查阅 `04-adaptation/`
- AI 工作流和禁止项：查阅 `05-rules/`
